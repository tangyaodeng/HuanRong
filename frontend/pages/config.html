创建特征表

表名命名规则：
  • 必须以字母或下划线开头
  • 允许字符：A-Z a-z 0-9 _ - $ # @
  • 长度限制：2-64个字符
  • 不能是MySQL保留关键字
  • 不能以短横线(-)开头或结尾
  • 不能包含连续的特殊字符
输入表名，系统会自动验证格式并提供建议

表结构预览

CREATE TABLE `新表名` (
  `UpdateDateTime` datetime DEFAULT NULL,
  `PointValue` float DEFAULT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci
