1. 查询模式图表：设备总功率优化趋势
图表说明：折线图展示“当前设备总功率”与“优化总功率”随时间变化趋势。
数据来源表：cooling_opt_parameters_total
图表系列	对应表字段	字段类型	说明
当前设备总功率	current_total_power	DECIMAL(8,2)	优化前的实际总功率（kW）
优化总功率	optimized_total_power	DECIMAL(8,2)	优化建议的总功率（kW）
X轴（时间）	updated_at 或 optimization_timestamp	TIMESTAMP	记录更新时间或优化时间戳
查询参数：start_date, end_date, limit

2. 寻优模式图表：迭代寻优过程
图表说明：展示最近一次优化迭代中所有温度‑功率组合下的总功率、主机功率、冷却塔功率、冷却泵功率分布。
数据来源：Redis 缓存（非数据库表）
Redis Key	数据结构	说明
HuanRong:cooling_opt:latest_iteration	JSON	由调度器 CoolingTempSchedulerTotal 在每次优化后写入
字段说明：
字段	类型	说明
index	int	按总功率降序的排名（1为最优）
cooling_inlet_temp	float	冷却水供水温度（℃）
cooling_return_temp	float	冷却水回水温度（℃）
delta_temp	float	供回水温差（℃）
host_power	float	主机总功率（kW）
cooling_pump_power	float	冷却泵总功率（kW）
cooling_tower_power	float	冷却塔总功率（kW）
total_power	float	设备总功率（kW）
power_diff	float	相比当前值的功率变化（负为节能）
power_diff_percent	float	功率变化百分比（负为节能）
